# asir-syntax-highlight
Asir用のVSCodeで使える拡張機能です。  

## 機能一覧
- コードの色分け
- 括弧補完
- 定義ジャンプ
- コード補完

## 使い方
Extension → Install from VSIX を選択し、vsixファイルを選んでください。
